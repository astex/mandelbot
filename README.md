# Squeak

A terminal emulator and IDE-like environment built for agentic development.

## Design Philosophy

- **Minimal opinionated configuration** — sensible defaults, few knobs
- **Effortless theming** — pick a theme and go, no tweaking required
- **Agentic development first** — built from the ground up for AI-assisted workflows

## Tech Stack

- **Rust** — core language
- **Qt via cxx-qt** — UI framework

## Building

Requires [rustup](https://rustup.rs/). The pinned toolchain will be installed automatically.

```sh
cargo build
```

## Status

Early exploration / hobby project.
